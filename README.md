# weimumu后端

media app(微信小程序\流媒体)
前端使用uni-app

## 技术栈

技术 | 名称                       
----|----
Spring Boot | 容器+MVC框架
Spring Security | 认证和授权框架
MyBatis PLUS | ORM框架  
MyBatisGenerator PLUS | 数据层代码生成  
Swagger2-UI | 文档生产工具
Elasticsearch | 搜索引擎
RabbitMq | 消息队列
Redis | 分布式缓存
Docker | 应用容器引擎
Druid | 数据库连接池
